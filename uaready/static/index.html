Validates the recipient against UAReady, then sends a real message via SMTPUTF8 (RFC 6531) if the SMTP server supports it. Requires uaready/sendmail.ini or UAREADY_* env vars.

What this checks
  • Unicode NFC normalisation before any comparison or encoding
  • Domain converted via IDNA2008 + UTS#46 (non-transitional)
  • Local part follows RFC 6532 (any non-ASCII Unicode allowed in mailbox)
  • RFC 5321 length limits — 64 octets local, 253 octets domain, 254 total
  • SMTPUTF8 requirement flagged when mailbox contains non-ASCII
  • Optional: MX lookup against the A-label (punycode) domain
